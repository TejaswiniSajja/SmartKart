# SmartKart 🛒

SmartKart is a modern e-commerce web application built using React.js and Tailwind CSS. It provides a seamless shopping experience with product browsing, cart management, and a clean, responsive user interface.

---

## 🌐 Live Demo

https://your-vercel-link.vercel.app

---

## 🚀 Features

* Browse products by category
* View detailed product information
* Add items to cart
* Update or remove items from cart
* Responsive design (mobile, tablet, desktop)
* Clean and modern UI

---

## 🛠 Tech Stack

* React.js
* Tailwind CSS
* JavaScript (ES6+)
* Context API (State Management)
* Axios (API handling)

## ⚙️ Installation & Setup

1. Clone the repository
   git clone https://github.com/your-username/SmartKart.git

2. Navigate to project folder
   cd SmartKart

3. Install dependencies
   npm install

4. Start the development server
   npm start
